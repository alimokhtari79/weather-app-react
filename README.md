## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/alimokhtari79/weather-app-react.git
```

2. Change the working directory

```bash
cd weather-app-react
```

3. Install dependencies

```bash
npm install
```

## ⚙️Running the app

```bash
$ npm start
```

🌟 You are all set!
